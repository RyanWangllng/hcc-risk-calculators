# 肝细胞癌术后辅助TACE治疗生存获益个体化预测模型

## 项目简介

这是一个基于Web的医学预测计算器，用于评估肝细胞癌根治性切除术后接受辅助经导管动脉化疗栓塞术(TACE)的生存获益。该模型可以帮助临床医生和患者进行个体化的治疗决策。

## 功能特点

- **个体化预测**：基于8个关键临床参数进行个体化风险评估
- **对比分析**：同时计算接受和不接受TACE治疗的预期结果
- **直观展示**：清晰展示生存时间和3年、5年生存率
- **响应式设计**：支持桌面和移动设备访问
- **现代化界面**：美观的用户界面和流畅的交互体验

## 临床参数

该预测模型需要输入以下8个临床参数：

1. **门脉高压** (是/否)
2. **大血管侵犯** (是/否)
3. **术前AFP水平** (<400 μg/L / ≥400 μg/L)
4. **微血管侵犯** (是/否)
5. **Child-Pugh分级** (A级/B级)
6. **切缘距离** (≥1cm / <1cm)
7. **肿瘤数量** (1个/2个/≥3个)
8. **最大肿瘤大小** (<5cm / 5-10cm / ≥10cm)

## 预测结果

系统将提供以下预测结果：

### 接受辅助TACE治疗
- 预期生存时间（月）
- 3年生存率（%）
- 5年生存率（%）

### 不接受辅助TACE治疗
- 预期生存时间（月）
- 3年生存率（%）
- 5年生存率（%）

### 净获益分析
- 生存时间差异（月）
- 3年生存率提升（%）
- 5年生存率提升（%）

## 使用方法

1. **打开网页**：在浏览器中打开 `index.html` 文件
2. **填写参数**：根据患者的临床资料选择相应的参数值
3. **计算结果**：点击"计算预测结果"按钮
4. **查看结果**：在右侧查看详细的预测结果和净获益分析

## 技术实现

### 前端技术
- **HTML5**：语义化标记和现代Web标准
- **CSS3**：响应式设计和现代视觉效果
- **JavaScript**：交互逻辑和计算算法

### 核心算法
- **风险评分模型**：基于临床因素计算综合风险评分
- **生存分析**：使用指数衰减模型计算生存时间和生存率
- **TACE效果评估**：根据患者特征评估TACE治疗效果

### 响应式设计
- 支持桌面、平板和手机设备
- 自适应布局和优化的触摸交互
- 跨浏览器兼容性

## 注意事项

### 重要声明
- 本预测模型仅供学术研究和临床参考使用
- 该计算器未指定推荐辅助治疗的特定阈值
- 最终治疗决策应该由临床医生和患者根据实际情况经过深思熟虑的讨论后做出
- 预测结果不能替代专业医学判断

### 数据来源
- 模型基于来自12家中国医院的2,514名患者数据构建和验证
- 算法参数基于已发表的临床研究进行调整

### 使用限制
- 仅适用于肝细胞癌根治性切除术后的患者
- 需要准确的临床病理数据输入
- 预测结果存在一定的统计学不确定性
