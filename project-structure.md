# 肝癌风险计算器网站项目结构

## 📁 项目文件结构

```
D:\self\
├── index.html                  # 主页 - 包含所有计算器的导航
├── main-style.css             # 主页样式文件
├── main-script.js             # 主页脚本文件
├── tace.html                  # TACE生存获益预测计算器（完整功能）
├── style.css                  # 子页面样式文件
├── script.js                  # 子页面脚本文件
├── hbv-hcc.html              # 乙肝相关性肝癌风险预测器（占位页面）
├── early-recurrence.html     # 肝癌早期复发风险预测器（占位页面）
├── morbidity-risk.html       # 肝癌手术患者发病率风险计算器（占位页面）
├── transfusion-risk.html     # 围手术期输血风险计算器（占位页面）
├── liver-failure.html        # 术后肝功能衰竭风险计算器（占位页面）
├── mortality-6m.html         # 术后6个月死亡率预测模型（占位页面）
├── complications-chb.html    # 乙肝患者术后并发症预测器（占位页面）
├── README.md                 # 项目说明文档
└── project-structure.md      # 项目结构文档（本文件）
```

## 🌟 功能特点

### 主页功能 (`index.html`)
- **8个计算器卡片**: 展示所有肝癌相关风险计算器
- **中英文双语**: 完整的语言切换支持
- **访问计数器**: 显示网站总访问量
- **响应式设计**: 支持桌面、平板、手机设备
- **卡片悬停效果**: 美观的交互动画
- **键盘快捷键**: 数字键1-8快速访问对应计算器

### TACE计算器 (`tace.html`)
- **完整功能**: 8个临床参数的生存获益预测
- **智能算法**: 基于风险评分的计算模型
- **双语界面**: 中英文实时切换
- **返回导航**: 一键返回主页
- **表单验证**: 智能输入验证和提示

### 占位页面
- **统一设计**: 所有占位页面保持一致的视觉风格
- **Coming Soon**: 明确显示开发状态
- **功能预览**: 展示每个计算器的计划功能
- **导航便利**: 返回主页和语言切换按钮

## 🎨 设计特色

### 视觉设计
- **现代化界面**: 渐变背景、卡片式布局

### 平板端 (768px-1024px)

- 保持所有功能
- 优化触摸交互

### 手机端 (<768px)
- 单列垂直布局
- 大按钮设计
- 简化界面元素

## 🔧 技术实现

### 前端技术栈
- **HTML5**: 语义化标记
- **CSS3**: 现代样式特性
- **JavaScript**: 原生ES6+语法
- **无依赖**: 不依赖外部库

### 核心功能
- **语言切换**: 基于data属性的双语系统
- **本地存储**: localStorage保存用户偏好
- **动画系统**: CSS3动画配合JavaScript
- **模块化**: 分离主页和子页面逻辑

## 🚀 部署说明

### 本地运行
1. 直接打开 `index.html` 文件
2. 或使用本地服务器（如Live Server）

### 生产部署
1. 上传所有文件到Web服务器
2. 确保服务器支持静态文件托管
3. 配置MIME类型（如需要）
